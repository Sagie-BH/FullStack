Flexbox Examples




Example 1: Flex Container with align-items and flex-direction

The align-items property controls how flex items align along the cross-axis of the container. Here, we have different options commented out to see the effects:
- align-items: flex-start; aligns items at the start of the cross-axis.
- align-items: flex-end; aligns items at the end of the cross-axis.
- align-items: center; centers items along the cross-axis.
- flex-direction: column; changes the main-axis to vertical.

Example 2: Flex Container with justify-content

The justify-content property determines how flex items distribute along the main-axis of the container. Here, we have different options commented out to see the effects:
- justify-content: flex-start; aligns items at the start of the main-axis.
- justify-content: flex-end; aligns items at the end of the main-axis.
- justify-content: center; centers items along the main-axis.
- justify-content: space-around; evenly distributes space around items along the main-axis.
- justify-content: space-between; evenly distributes space between items along the main-axis.

Example 3: Flex Container with flex-wrap

The flex-wrap property determines whether flex items should wrap onto multiple lines if they can't fit in a single line. Here, we use flex-wrap: wrap; to wrap items to a new line.
The justify-content: space-between; property evenly distributes the space between items along the main-axis, creating a pleasing layout.
